🌱 Project Spark: My 4-Year Game Plan

Year 1 — Plant the Seed

  • ☐ Choose one idea
  • ☐ Add branding
  • ☐ Build version 0.1
  • ☐ Learn skills (coding, design)
  • ☐ Tell 3 friends

Year 2 — Grow Roots

  • ☐ Build a real demo
  • ☐ Add features
  • ☐ Get 5 users
  • ☐ Learn basic marketing

Year 3 — Make It Real

  • ☐ Redesign or rebrand
  • ☐ Reach 100 users
  • ☐ Start monetizing
  • ☐ Optional: build a team

Year 4 — Level Up or Let Go

  • ☐ Decide on next steps
  • ☐ Launch final version or pivot
  • ☐ Add to portfolio/resume
  • ☐ Reflect on everything you learned
